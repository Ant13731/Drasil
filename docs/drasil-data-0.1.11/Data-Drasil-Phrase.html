and_ :: (NamedIdea c, NamedIdea d) => c -> d -> NP #

Creates an NP by combining two NamedIdeas with the word "and" between their terms. Plural is defaulted to (phrase t1) "of" (plural t2)

and_' :: (NamedIdea c, NamedIdea d) => c -> d -> NP #

Same as and_ combinator, except phrase default of second term is plural instead of phrase

for :: (NamedIdea c, NamedIdea d) => c -> d -> Sentence #

Inserts the word "for" between the titleized versions of two terms

for' :: (NamedIdea c, Idea d) => c -> d -> Sentence #

Similar to for, but uses titleized version of term 1 with the abbreviation (if it exists, phrase otherwise) for term 2

for'' :: (c -> Sentence) -> (d -> Sentence) -> c -> d -> Sentence #

Similar to for, but allows one to specify the function to use on each term before inserting for. For example one could use for'' phrase plural t1 t2
